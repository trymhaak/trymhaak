# Trym Håkansson

Consultant at [Spirhed](https://github.com/Spirhed), working on Microsoft security and Azure hybrid platforms: Defender XDR, Microsoft Sentinel, Entra ID, Azure Local, and datacenter modernization.

I build small, practical tools that remove friction in security operations, and I spend a lot of time on agentic developer workflows and MCP servers.
